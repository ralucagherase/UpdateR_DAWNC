Update R

Acest raport descrie curentul progress la proiectul numit „Update R ” care va fi evaluat la disciplina Dezvoltarea Aplicatiilor Web la Nivel de Cleint promoția 2017-2018. Participantele la crearea acestui proiect sunt Gherase Raluca si Andrian Diana.

1.Introducere

Se doreşte realizarea unui instrument Web (i.e. o extensie de navigator Web) care permite indicarea schimbărilor ce au survenit în cadrul unei pagini Web faţă de vizita precedentă. Se vor oferi posibilităţi de configurare (e.g., lista URL-urilor care nu vor fi luate în consideraţie, modul de redare a zonelor actualizate ale documentului, întreruperea temporară a funcţionalităţii instrumentului etc.), cu suport pentru export/import al datelor de configurare la nivel local sau recurgând la un serviciu de stocare în "nori" precum Dropbox, OneDrive etc.

3.Tehnologii utilizate

Proiectul utilizează următoarele tehnologii: HTML5 - este un limbaj pentru structurarea și prezentarea conținutului pentru World Wide Web, o tehnologie nucleu pentru Internet propusă inițial pentru software-ul Opera.

CSS - (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML. Stilurile se pot atașa elementelor HTML prin intermediul unor fișiere externe sau în cadrul documentului, prin elementul "style" și/sau atributul style. CSS se poate utiliza și pentru formatarea elementelor XHTML, XML și SVGL

SASS(.scss)- Este un pre-procesor CSS (extensie CSS) care permite utilizarea unui limbaj mult mai puternic si inteligent decat simplul CSS. Odata ce este compilat acesta va genera fisierul CSS care va putea fi folosit pentru aplicatia web in mod normal. Alternativele la Sass sunt: Less si Stylus si acestea fiind utilizate pentru acelasi scop.

Javascript - este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor.Este folosit mai ales pentru introducerea unor funcționalități în paginile web, codul Javascript din aceste pagini fiind rulat de către browser. Limbajul este binecunoscut pentru folosirea sa în construirea siturilor web, dar este folosit și pentru acesul la obiecte încastrate (embedded objects) în alte aplicații. A fost dezvoltat inițial de către Brendan Eich de la Netscape Communications Corporation sub numele de Mocha, apoi LiveScript, și denumit în final JavaScript.

NodeJS - Node.js este un mediu open-source, cross-platform JavaScript de execuție pentru executarea codului de server JavaScript. Din punct de vedere istoric, JavaScript a fost folosit în primul rând pentru scripting pe partea clientului, în care scripturile scrise în JavaScript sunt încorporate într-un HTML al unei pagini Web, pentru a fi executate de utilizator in locul clientului in browser. Node.js permite JavaScript-ului să fie utilizat pentru scripting-ul de partea serverului și execută scripturile de pe server pentru a produce conținut dinamic al paginii web înainte ca pagina să fie trimisă în browser-ul web al utilizatorului. În consecință, Node.js a devenit unul dintre elementele fundamentale ale paradigmei "JavaScript", care permite dezvoltării aplicațiilor web să se unească în jurul unui singur limbaj de programare, în loc să se bazeze pe o altă limbă pentru scrierea scripturilor de pe server.

Firebase - este o platformă de dezvoltare a aplicațiilor mobile și web dezvoltate de Firebase, formata în 2011, achiziționată apoi de Google în 2014. Ofera suport pentru aplicatii mobile, web , baza de date si hosting.

4. Structura

Extensia
• Va conține doua butoane, un "intrerupator" care va memora sau nu pagina selectata in cardul tab-ului si un botun care va duce pe pagina cu setarile aplicatiei
• Va conține numarl schimbarilor ce apar in cadrul tabului pe care noi ne aflam si este deja supravegheat
• Va conține titlul aplicatiei
• Va conține un selector de culoare bazat pe jscolor.

Pagina Settings
• Va conține 2 containere in care vom regasi istoricul schimbarilor din cadrul paginilor monitorizate si unul in care vom vedea paginile pe care le monitorizam cu optiunea de remove pagina, in cazul in care nu mai dorim monitorizarea acesteia
• Va conține 4 butoane: unul pentru adaugarea de noi pagini ce deschide un form peste pagina, unul in care putem alege un fisier de configurare, unul de import ce salveaza din fisierul de configurare schimbarile pe pagina si unul de export care exporta toate informatiile de pe pagina
• Va contine de asemenea si un colorpicker de forma jscolor in care vom seta culoarea cu care vom vedea schimbarile pe paginile monitorizate
